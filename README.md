# Projeto de Gerenciamento de Pedidos
Este projeto é uma API para gerenciamento de pedidos e itens de pedidos, desenvolvida em ASP.NET Core com Entity Framework Core e Angular para o front-end. 

## API Endpoints
### Pedidos
GET /api/pedido - Retorna todos os pedidos. <br>
GET /api/pedido/{id} - Retorna um pedido específico pelo ID. <br>
POST /api/pedido - Cria um novo pedido. <br>
PUT /api/pedido/{id} - Atualiza um pedido existente. <br>
DELETE /api/pedido/{id} - Deleta um pedido específico. <br>
